# The HTML App of JGiven and JsGiven

The interactive HTML5 App of [JGiven](https://jgiven.org) and [JsGiven](https://jsgiven.org).

Uses the power of AngularJS, Foundation, and Font Awesome

## Build

```
yarn install
yarn build
yarn test
```
